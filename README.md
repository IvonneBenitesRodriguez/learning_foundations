## A Review of JavaScript foundations
This repo has been made as a review of main foundations of programming language JavaScript, concepts such as variables and its scope, iterators and its different types of iterations, conditionals, functions, arrays, objects are present in this repo.<br/>
Futhermore, some easy to understand examples and interface implementations will be added.<br/>
The literature is based on Data Structures and Algorithms Book and online resources.<br/>
