## Variables Types<br/>
In JavaScript, we have a primitive operator named <strong> typeof</strong> that returns always the type of variable that has been used.<br/>
Example:<br/>
var is20 = false; <br/>
typeof is20; // boolean<br/>
var age = 19;<br/>
typeof 19; // number<br/>
var lastName = "Bae";<br/>
typeof "Bae"; //string <br/>
var fruits = ["apple", "banana", "orange"];
typeof fruits; //<strong> special type of objects that are arrays</strong><br/>
var person1 = {firstName: "Maria", lastName: "Ruiz"};<br/>
typeof person1;// this is an object <br/>
var nullVar = null;<br/>
typeof nullVar; // object<br/>
var blank; <br/>
typeof blank; // undefined<br/>
