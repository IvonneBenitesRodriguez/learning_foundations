### Functions
- Organize code.
- Promote reusability.
- Improve maintainability.
<br/><br/>
Syntax:<br/><br/>
function myname (name) {<br/>
    console.log("Hello! ", name);<br/>
}<br/><br/>
the function will create a reusable piece of code that greets a person by their name. The parameter used is name. This parameter is used as a placeholder for a value that will be provided when the function is called.
<br/><br/>
To execute the code we only write:<br/><br/>
myname('Peter');

A function can return also a value:<br/><br/>
function sum(num1, num2){
    return num1 + num2; 
}<br/>
A function can return a value using the <strong> return keyword </strong> followed by an expression.<br/>
