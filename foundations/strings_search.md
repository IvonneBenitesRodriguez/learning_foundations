### Strings Search:
To find a specific string within a string, this command is used:<br/><br/>
<strong>.indexOf(searchValue[,fromIndex]);</strong>
<br/><br/>
Example:<br/>
'Peru'.indexOf('Peru');  // returns 0 <br/><br/>
'Peru Tours'.indexOf('Peruvian');//returns -1<br/><br/>
'Peru Air'.indexOf('Air',0);  // returns 5 <br/><br/>
To <strong>find a string which starts with a specific input or ends with a specific input,</strong> use this command:<br/><br/>
<strong>.startsWith( )</strong><br/>
Example:
'Peru Tours'.startsWith('Peru');// returns true <br/><br/>
<strong>.endsWith( )</strong><br/>
'Peru Tours'.endsWith('Tours');// returns true <br/><br/>
